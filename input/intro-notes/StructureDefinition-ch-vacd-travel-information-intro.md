### Scope and Usage

This Condition profile ist used to define travel informations to be sent with an [CH VACD Immunization Recommendation Request Message](StructureDefinition-ch-vacd-recommendation-request-message.html) to a CDS System to receive [ImmunizationRecommendations]() about the travel destination.
