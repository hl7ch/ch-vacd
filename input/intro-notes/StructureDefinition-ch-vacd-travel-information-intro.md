### Scope and Usage

This Condition profile is used to define travel informations to be sent with an [Immunization Recommendation Request Message](immunization-recommendation-request-message.html) to a CDS System to receive a [Immunization Recommendation Response Message](immunization-recommendation-response-message.html) with [Immunization Recommendations](StructureDefinition-ch-vacd-immunization-recommendation.html) in context to the travel destination.
