### Scope and Usage

This extension is used to reference a [CH VACD Medication For Immunization resource](StructureDefinition-ch-vacd-medication-for-immunization.html) for documenting the applied vaccine product in reference to the [CH VACD Immunization](StructureDefinition-ch-vacd-immunization.html). The Immunization resource is only referencing by Immunization.vaccineCode the vaccination given to the patient. The effective used product identifieable by the GTIN code has to be referenced using this extension to the Medication resource.
