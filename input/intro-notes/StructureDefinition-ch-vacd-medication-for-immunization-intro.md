### Scope and Usage

This profile defines the Medication resource in context of the documentation of a vaccination event.
In the [Immunization](StructureDefinition-ch-vacd-immunization.html) profile is the applicated vaccination (Immunization.vaccineCode) defined and not the used product.
The used Product may to be documented using this profile and will be referenced in the Immunization reference by using the extension [CH VACD Extension Immunization Medication Reference ](StructureDefinition-ch-vacd-ext-immunization-medication-reference.html).
