
Since this IG was originally created as a specification for a document-based exchange format, it did not contain any specific documentation on the FHIR RESTful API.
Nevertheless, the resource profiles defined in this IG can be used in the context of a server with FHIR RESTful API. This also includes uploading and downloading a ch-vacd document.

See here a possible [Capability Statement](CapabilityStatement-ch-vacd-api-capability-statement.html) for a FHIR server.
